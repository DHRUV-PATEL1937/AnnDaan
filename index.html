We Are The Best Redistribution Network in Town

FoodLink is a technology-first initiative that bridges food waste and food insecurity. Donators request pickups; NGO admins assign riders; volunteers verify quality and distribute meals to reported hot-spots. Everything is auditable, safe, and lightning fast.

Vision

A city where no safe meal gets wasted and no person sleeps hungry.

Mission

Use design and data to move good food to the right place at the right time.

Who uses FoodLink?

Event hosts, restaurants, canteens, campuses, shelters, and volunteer groups.

We Provide the Best, Safest Flow

From pickup request to verified distribution—clear, accountable steps.

1

Donator creates a pickup

Location, food type, quantity—submitted in seconds via web or mobile.

2

NGO assigns a rider

Admins see pending requests and match the closest available volunteer.

3

4

Distribution & reporting

Meals go to mapped hot-spots submitted by community reporters.

Modules Overview

Clear responsibilities with a friendly UI for each role.

Donator

  • Create food pickup request (location, quantity, type)
  • Track status and get notifications

NGO Admin

  • View and assign pickup requests
  • Monitor volunteer activity and distribution

Rider / Volunteer

  • Receive nearby pickups and accept/decline
  • Verify food quality & update status

Reporter

  • Send alerts about people in need
  • Help NGOs plan distribution routes

Developer-Ready API

Simple, secure endpoints for automation and integrations.

# Donator — Create a new pickup POST /api/requests/pickup Content-Type: application/json { "location": "28.6139, 77.2090", "quantity": 60, "foodType": "Veg Meals", "notes": "Wedding leftovers" } # Rider — Get nearby pickups GET /api/requests/nearby?lat=28.61&lng=77.20&radius=5km # Volunteer lifecycle POST /api/volunteer/accept { "requestId": "R-1029" } POST /api/volunteer/verify { "requestId": "R-1029", "score": 4, "checklist": ["hot", "sealed"] } # Reporter — Flag a hotspot POST /api/alerts/needy { "coords": "28.64,77.20", "description":"Under flyover" } # NGO Admin — queue GET /api/admin/requests?status=pending

Frequently Asked Questions

Is FoodLink free to use?

Yes. We operate via grants and community support. Donations help us run servers and training.

What if the food is unsafe?

Riders rate quality via checklist. Low scores halt distribution and notify admins automatically.

Which maps do you support?

Google Maps or OpenStreetMap—selected by the partner city. Both support offline-tolerant routing.

Can I integrate my campus canteen?

Yes. Use the REST API to automate recurring pickups and add staff as riders.
